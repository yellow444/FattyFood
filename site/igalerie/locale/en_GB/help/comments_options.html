Comment moderation

By checking this option, the comments will not be visible on the gallery as long as you have not validated them. Moderated comments are disabled (status "pending") and you must activate them in the section "Comments / Images".

Note that this option is ignored when the feature "Members area" is enabled. In this case, to moderate comments you must select the "On hold" option for the group permission "Add comments".

Icon set for smilies

To add a new icon set, you must create in the directory "images / smilies" a directory whose name does not contain spaces, accented or special characters, then add your icon files and then create a file "icons.php "used to make the correspondence between the code for each smiley and the file name of the icon, using the following example :

<?php
$smilies[':???:'] = 'confuse.png';
$smilies[':cool:'] = 'cool.png';
$smilies[':cry:'] = 'cry.png';
$smilies[':eek:'] = 'eek.png';
?>

You are free to add the codes and the number of smilies you want.

Anti-flood

The anti-flood adjusts, after sending a comment, how long a user can not post another comment.

Maximum number of URLs

By totally banning the presence of URL (0) or greatly limiting it (1), you will block a considerable amount of spam. Indeed, most spam messages contain one or more URLs to other sites.
