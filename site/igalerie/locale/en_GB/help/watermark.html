Font

You can add new fonts in the directory "fonts" of iGalerie.

Only True Type fonts with the extension .ttf are accepted.

Transparency

Transparency must be an integer between 0 (no transparency) and 127 (completely transparent).

Proportional size

The proportional size indicates the percentage of image width to be occupied by the watermark.

Note that this value is approximate.

Watermark image

The images watermark must be placed in the directory "images / watermark". They should be formatted JPEG , GIF or PNG , with the extension .jpg,.gif or.png. The file name of the images can only contain dashes (-), underscores (_) and alphanumeric characters (az and 0-9).

Quality

The quality of the image created by GD must be an integer between 0 (very poor) to 100 (best quality). The higher the quality, the greater the weight of the image increases.
