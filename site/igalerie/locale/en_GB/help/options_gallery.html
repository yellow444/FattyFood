Banner

You can place a banner instead of the title (but the title of the gallery will still be used in the title bar of the browser, as well as replacement of the banner should it not be shown ). For this you need to place an image of format JPEG, GIF or PNG,and with the extension. jpg,. gif or. png in the folder "images / banners". The file name of the images can only contain dashes (-), underscores (_) and alphanumeric characters (az and 0-9).

URL rewriting

The URL rewriting allows for shorter URL by suppressing the "?q=" off the URL. That is to say that instead of having URLs of this type :

http://www.igalerie.org/demo/?q=album/3-nature

The URL rewriting will give you the following :

http://www.igalerie.org/demo/album/3-nature

For URL rewriting to work, and before enabling the feature, you must add, to the .htaccess file located in the directory "iGallerie", the following lines (if they are not already present) :

<IfModule mod_rewrite.c>
	RewriteEngine on
	RewriteBase /
	RewriteCond %{REQUEST_FILENAME} !-f
	RewriteCond %{REQUEST_FILENAME} !-d
	RewriteRule (.*) index.php?q=$1 [QSA,L]
</IfModule>

Languages

Uninstall languages

Note that it is impossible to uninstall the default language.

Warning ! If you uninstall a language all your localized texts in that language will be deleted permanently !

Dates

For the custom date format, you must enter a date format compatible with the function PHP strftime().

GD parameters

Transparency management

You must select this option if you have images of format GIF or PNG and want to maintain the transparency of your images for thumbnails and intermediate size pictures.

Adding Images

Add all files to the report

If this option is enabled, the scan report by FTP will mention all files that have been rejected, not just images.

Updating images

If this option is enabled, the scan via FTP will update the edited images. If for example you replace an image by changing its size, it will be updated in the database. Note that this option slightly increases the duration of scan, due to the fact that it is necessary to check each image inside the album being updated.

Choose a new thumbnail for category updates

If this option is enabled, a new thumbnail will be chosen for albums and categories updated. The image selected as the thumbnail is not random, it is always the last image added during scanning. This option affects as much the scan by FTP than the adding of images by HTTP (ie sending of pictures through the browser).
