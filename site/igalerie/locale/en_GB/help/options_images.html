Direct links to images

By enabling this option, images will be displayed directly by clicking their respective thumbnail, ie they will not be displayed in a gallery page.

Therefore, this option disables some features only accessible from the page "images", such as comments, votes and favorites.

This option is useful if you want to display your images in a new window or in a script style "lightbox" for example (for this, we must modify the code Markup HTML of the theme).

Intermediate size images with GD

This option will create images of intermediate size if images exceed the specified size. If you choose this feature, visitors will not be able to choose how to re-size images.

The difference with HTML resizing is that the images are of better quality. HTML resizing only re-size the original images inside the browser.

For quality scaled images by GD, you must specify a number between 0 (worst quality) to 100 (best quality).
