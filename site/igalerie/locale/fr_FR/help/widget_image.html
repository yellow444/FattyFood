Bloc image

Le "bloc image" est un widget permettant d'afficher une ou plusieurs vignettes d'images fixes ou aléatoires.

Identifiant des images et albums

Dans la section "Albums", il est indiqué "album 2", "album 3", etc. sous le statut des albums ("publié", etc.) à droite. L'identifiant d'une catégorie, album ou image est ce nombre, que l'on retrouve également toujours dans le paramètre d'URL lorsque l'on affiche un album ou une image : "?q=album/2" ou "?q=image/10" pour le panneau d'administration et "?q=album/2-titre-de-l-album" ou "?q=image/10-titre-de-l-image" pour la galerie.
