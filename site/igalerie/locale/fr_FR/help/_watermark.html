Fonte

Vous pouvez ajouter de nouvelles fontes dans le répertoire "fonts" d'iGalerie.

Seules les fontes TrueType avec l'extension .ttf sont acceptées.

Transparence

La transparence doit être un nombre entier compris entre 0 (aucune transparence) et 127 (transparence complète).

Taille proportionnelle

La taille proportionnelle indique le pourcentage de la largeur de l'image que doit occuper le filigrane.

Notez que cette valeur est approximative.

Qualité

Pour ajouter un filigrane sur vos images, iGalerie doit créer de nouvelles images. Entrez un nombre entre 0 (très mauvaise qualité) et 100 (meilleure qualité possible) pour définir la qualité des images générées. Plus la qualité est grande, plus le poids de l'image augmente.
