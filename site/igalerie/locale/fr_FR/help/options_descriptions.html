Modèles de description

Les modèles de description permettent d'appliquer des éléments commun (texte ou HTML) pour toutes les descriptions des catégories et images. Si vous souhaitez afficher, en plus ou à la place de la description de chaque catégorie ou image, le même texte ou le même code HTML, cela est possible grâce aux modèles de description. L'intérêt du modèle de description est que vous disposez en outre de plusieurs "variables" contenant des informations des catégories et des images, ainsi que des structures conditionnelles qui permettent de tester la présence de certaines de ces informations.

Les variables disponibles sont : {DESCRIPTION}, {ID}, {PLACE}, {TITLE}, {URL} pour les catégories et images, ainsi que {FILENAME}, {HEIGHT} et {WIDTH} pour les images uniquement. Les structures conditionnelles disponibles sont {IF(DESCRIPTION)}...{ENDIF(DESCRIPTION)} et {IF(PLACE)}...{ENDIF(PLACE)}.

Voici un exemple de modèle de description pour catégories :

{IF(DESCRIPTION)}<a href="javascript:;" rel="category_description_{ID}" class="show_parts js_link">Lire la description</a>
<span id="category_description_{ID}" style="display:none">
{DESCRIPTION}</span>{ENDIF(DESCRIPTION)}
